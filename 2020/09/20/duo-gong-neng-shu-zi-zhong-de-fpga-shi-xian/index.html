多功能数字钟的FPGA设计与实现

姓名：吴栋

班级：电信1803

学号：U201813381

合作者：吴雨暄

实验目的

  1. 熟悉并掌握利用Verilog建模描述具体数字电路的方法
  2. 熟悉FPGA开发流程，Vivado软件的使用和Nexys 4DDR开发板的应用
  3. 了解数字钟的功能要求及设计方法

实验任务

  1. 完成多功能数字钟的总体设计和模块划分
  2. 在Vivado平台上完成多功能数字钟的Verilog代码开发，以及编写引脚约束的代码文件
  3. 生成bitstream并下载 到Nexys 4DDR开发板上，调试功能

实验要求

  1. 准确计时，利用七段数码管以数字形式显示时、分、秒的时间，且可以切换小时显示（12/24）模式，24小时计时规律为0-1-2-…-23,12小时计时规律为12-01-02-…-11，且12小时制时有对应的标志位用以区分上下午。
  2. 可以手动校时、校分、校秒。
  3. 可以设置闹钟，闹钟时间精确到分钟，并且闹钟响应期间可以手动关闭。
  4. 具有秒表功能，计时精度为10ms，支持手动复位和暂停/继续。
  5. 具有基于12小时制的整点报时功能。

实验设计方案

模式设计和模块划分

基于实验要求，我们为数字钟划分了4种工作模式，分别是常规显示模式、调时模式、设定闹钟模式、秒表模式。我们选用两位开关sw0、sw1切换工作模式：

  • sw0=0 sw1=0 : 常规显示模式
  • sw0=0 sw=1 : 调时模式
  • sw0=1 sw=0 : 设定闹钟模式
  • sw0=1 sw=1: 秒表模式
常规显示模式

当数字钟处于常规显示模式时，数字钟能准确计时，并利用七段数码管显示当前时、分、秒时间。

且我们选用开关sw15作为小时显示（12/24）模式的控制开关，开关闭合时为12小时制，反之为24小时制，当显示模式为12小时制时，我们通过最高位的数码管显示A、P区分上下午，当显示模式为24小时制时，最高位数码管为常熄，显示模式是可以任意切换的。

当时间到达我们设定的闹钟时间时，通过时间显示数码管的闪烁代表闹铃的响起，我们可以通过BTNC按键手动关闭闹铃，也可以等待一分钟，闹铃会自动关闭。

当时间到达整点时，我们通过两位RGB LED的三种颜色轮番闪烁区分上下午，其中LD16闪烁时代表上午，LD17闪烁时代表下午，并利用12位LED灯的长明区分具体小时，整点报时功能只持续6秒，6秒时间一过，普通LED和RGB LED都全部熄灭。

显示时间可以通过调时模式进行校正，闹钟时间可以通过设定闹钟模式进行设定。

调时模式

调时模式为我们的数字钟校正计时时间，当数字钟的工作模式切换到调时模式时，利用七段数码管的前两位显示AL，表明当前工作模式为调时模式，我们将常规显示模式的当前时间进行传入，在当前时间的基础上进行调时，为尽可能提高调时效率，我们可以直接对秒个/时位、分个/时位、小时位进行调试，我们通过对应位闪烁表明该位位当前调时位，通过BTNL、BTNR按键左右滑动选择调时位，通过BTNU、BTND按键进行增减调动，且调时时，我们也可以任意切换选择我们喜欢的显示（12/24）模式，调时完毕后，我们通过按下BTNC按键就可以用调好的时间去校正常规显示模块的显示计时时间了。

设定闹钟模式

设定模式为我们的数字钟设定闹钟时间，当数字钟的工作模式切换到设定闹钟模式，我们可以在上次设定的闹钟时间的基础进行调时，类似调时模式，通过对应位闪烁表明该位位当前调时位，可以通过BTNL、BTNR按键左右滑动选择调时位，通过BTNU、BTND按键进行增减调动，也可以任意切换我们喜欢的显示（12/24）模式，不过我们闹钟时间是精确到分钟的，所以我们调时位只有小时位、分个/十位，当我们从设定闹钟模式切换到数字钟其他工作模式时，设定好的闹钟时间就已经自动更新了。

秒表模式

秒表模式提供10ms精度的计时功能，计时时间范围为00:00:00到59:59:99,我们通过BTNC按键启动 /暂停/继续计时，通过BTND按键清零复位。

顶层结构设计

基于各模式间的耦合关系，由于常规显示模块的显示时间和闹钟时间需要调试模块和设定闹钟模式校正设定，调时模式也需要将常规显示模块的当前时间进行传入，而秒表模式相对独立，所以我们在上层模块设定这样的引用关系：

秒表频率发生器

秒表计时需产生周期为10ms频率为100HZ的时钟，此外还需满足秒表能够暂停/继续的功能。故而该模块主要功能为通过计数器对Nexys 4DDR开发板上的100MHZ频率分频，产生100HZ的扫描频率。在此基础上，增加一标志位stop，检测暂停按键上升沿改变stop的状态，对stop进行翻转。在计数器中增加一次判断，只有当stop为1时才计数，否则保持，从而停止秒表时钟的产生，以达到暂停的效果。当按下暂停按键时，stop翻转为0，计时停止；再次按下暂停按键后,stop重新翻转为1，继续计时。

module timediv(
    input clk_100m,
    input timer_EN,
    input pause,
    output reg clk_out
    );
    reg [30:0] count_div;
        parameter clk_freq = 100000000    //原时钟频率
    parameter  out_freq=100;    //输出频率
    reg stop;
    initial
    begin
    clk_out <= 1'b0;
    count_div <= 0;
    stop <= 1; 
    end

    always @(posedge pause)
    begin
        if(timer_EN)
            stop <= ~stop;
    end

    //计数翻转
    always @(posedge clk_100m)
    begin
        if((! stop) && timer_EN)
        begin
            if(count_div<(clk_freq/(2*out_freq)-1))
                count_div <= count_div + 1'b1;
            else begin
                count_div <= 0;
                clk_out <= ~clk_out;
            end
        end
    end
endmodule

秒表计数器

秒表显示共有6位，计时范围为00:00:00至59:59:99低3位与第5位为0到9计数模十，第4位与最高位为0~5计数模六，且需要实现复位的功能。故而该模块功能为同步可变模长计数以及异步清零。将6个计数器进行级联，其中每个计数器的计数使能信号由前级计数使能信号和前级是否达到最高位（5或者9）的信号相与获得。当清零信号有效时，所有计数器异步清零实现秒表的复位功能。

module tmicnt(
    input clk,
    input timer_EN,
    input [4:0] mod,
    input reset,
    input en,
    output reg [3:0] Q
    );
    initial
    begin
        Q <= 0;
    end

    always @(posedge clk or posedge reset)
    begin
        if(reset)    //置零
        begin
            if(timer_EN)
                Q<=0;
        end
        else if(~en)    //保持
            Q <= Q;
        else    //计数
            Q <= (Q + 1) % mod;
    end
endmodule

  • 计数器(小时)**，同样是同步计数，异步置数。在按下confirm后若当前处于调时模块则将设置的小时值赋给计数器，不同的是在置数后需要统一将24/12进制的小时数统一转化为24小时进制数，在正常计数时统一采用24小时进制计数，输出时再根据当前的显示模式进行进制换算，由24小时制转换成24/12小时制
`include"para.v"
module count_Hour(
    input clk,en,
    input Mode,
    input select_EN,
    input confirm,
    input [3:0] H_set,L_set,Mode_tag_set,
    output reg [3:0] CntH,CntL,Mode_tag
    );
    reg [3:0] CntH_temp,CntL_temp;
    reg [4:0] tmp;
    parameter ten=4'd10;
    reg [5:0] Cnt;


    always @(posedge clk or posedge confirm)
    begin
        //现将12/24进制数时间统一按规矩转化为24进制数
        if(confirm)
        begin
            if(select_EN)
            begin
                CntH_temp = H_set;
                CntL_temp = L_set;
                tmp = CntH_temp * ten + CntL_temp;
                if(Mode_tag_set == 4'd11)
                begin
                    if (tmp == 5'd12)
                    begin
                         CntH_temp <= 4'd1;
                         CntL_temp <= 4'd2;
                    end
                    else
                    begin
                         CntH_temp <= (tmp+5'd12-(tmp+5'd12)%ten)/ten;
                         CntL_temp <= (tmp+5'd12)%ten;
                    end
                end
                else if(Mode_tag_set == 4'd10)
                begin
                    if(tmp == 5'd12)
                    begin
                         CntH_temp <= 4'd0;
                         CntL_temp <= 4'd0;
                    end
                end
            end
        end

        //保持
        else if(~en)
            {CntH_temp,CntL_temp} <= {CntH_temp,CntL_temp};

        //计数
        else
            if((CntH_temp>2)||(CntL_temp>9)||((CntH_temp==2)&&(CntL_temp>=3)))
                {CntH_temp,CntL_temp} <= 8'h00;
            else if((CntH_temp==2)&&(CntL_temp<3))
                 begin   CntH_temp<=CntH_temp; CntL_temp<=CntL_temp+1'b1;    end
            else if(CntL_temp==9)
                 begin CntH_temp<=CntH_temp+1'b1;  CntL_temp<=4'b0000;  end     //小时十位的计数
            else
                 begin CntH_temp<=CntH_temp;   CntL_temp<=CntL_temp+1'b1;    end
    end

    //将小时数、上下午标志转化为要求的输出模式
    always@(CntH_temp or CntL_temp or Mode)
    begin
        if(Mode==`Mode_24)
        begin
            CntH<=CntH_temp;
            CntL<=CntL_temp;
            Mode_tag<=4'd12;
        end
        else
        begin
            Cnt = CntH_temp * ten+CntL_temp;
            if((Cnt>=5'd1) && (Cnt<=5'd11))
            begin
                CntH<=CntH_temp;
                CntL<=CntL_temp;
                Mode_tag<=4'd10;
            end
            else if(Cnt==5'd0)
            begin
                CntH<=4'd1;
                CntL<=4'd2;
                Mode_tag<=4'd10;
            end
            else if(Cnt==5'd12)
            begin
                CntH<=4'd1;
                CntL<=4'd2;
                Mode_tag<=4'd11;
            end
            else
            begin
                CntH<=(Cnt-5'd12-(Cnt-5'd12)%ten)/ten;
                CntL<=(Cnt-5'd12)%ten;
                Mode_tag<=4'd11;
            end  
        end
    end
endmodule

手动调时（分/秒）

手动逐位校分/秒模块实现将传入的当前时间作为校时初值，并通过上下按键对当前位的数值改变。

  1. 设置校时初值：设置一标志位flag，每当调时使能select_EN无效，即处在非调时模块时，将标志位置1，开放置数使能。当进入调时模块时，调时使能和置数使能都有效，此时将传入的当前时间赋值给计数器，并将标志位flag置0关闭置数使能。直到退出调试模式再开放置数使能
  2. 按键调时：具体实现方式与左右按键选位类似，都是在高频时钟下设置标志位作为按键响应开关的同步电平检测，并在此基础上加了一次判断，只有当闪烁位置等于当前位置时才进行计数，实现逐位调时
module manua_lcount(
    input clk,
    input select_EN,
    input buttu,buttd,
    input [3:0] mode,
    input [3:0] flicker_pos,current_pos,
    input [3:0] set,
    output reg [3:0]Q
    );
    reg up_flag,down_flag;
    reg flag;
    initial
    begin
        up_flag <= 1;
        down_flag <= 1;
        flag <= 0;
    end

    always @(posedge clk)
    begin
        if(! select_EN)
            flag <= 1;
        if(select_EN && flag)   
        begin
            Q <= set;
            flag <= 0;
        end

        //
        if(select_EN)
        begin
            if(flicker_pos == current_pos)
            begin
                //设立标志位，防止一直调动
                if(! buttu)
                    up_flag<=1;
                if(! buttd)
                    down_flag<=1;
                //上调
                if(up_flag && buttu)
                begin
                    Q <= (Q + 1) % mode;
                    up_flag <= 0;
                end
                //下调
                if(down_flag && buttd)
                begin
                    if(Q == 4'd0)
                        Q <= mode - 1;
                    else
                        Q <= Q - 1; 
                    down_flag <= 0;
                end
            end
        end
    end
endmodule

手动调时（时）

手动校时部分的大体思路和逻辑与手动校分/秒类似，不同之处在于传入当前小时时要先进行进制转换，无论传入的是24小时制还是12小时制，统一转化为24小时制便于计数。此外还需实施检测显示进制的变化，因此增加了一个always检测小时高位，低位和进制，任何一项改变时根据当前的显示进制进行进制换算，输出相应的小时数以及进制标志。

`include"para.v"
module manual_count24(
    input clk,
    input select_EN,
    input Mode,
    input buttu,buttd,
    input [3:0] flicker_pos,current_pos,
    input [3:0] Mode_set,
    input [3:0]H_set,L_set,
    output reg [3:0]Modetag,H,L
    );
    reg up_flag,down_flag;
    reg flag;
    reg [4:0] tmp;
    reg [4:0] tmp_2;
    parameter ten=4'd10;
    reg [3:0] Modetag_temp,H_temp,L_temp;
    initial
    begin
        flag <= 0;
    end

    always @(posedge clk)
    begin
        if (! select_EN)
            flag <= 1;
        if(select_EN && flag)
        begin
            Modetag_temp = Mode_set;
            H_temp = H_set;
            L_temp = L_set;
            //统一转化为24小时制时间
            if (Modetag_temp == 4'd10)
            begin
                tmp = H_temp * ten + L_temp;
                if(tmp == 5'd12)
                begin
                    H_temp <= 4'd0;
                    L_temp <= 4'd0;
                end
                Modetag_temp <=4'd12;
            end
            else if (Modetag_temp==4'd11)
            begin
                tmp = H_temp * ten + L_temp;
                if (tmp == 5'd12)
                begin
                    H_temp <= 4'd1;
                    L_temp <= 4'd2;
                end
                else
                begin
                    H_temp <= (tmp+5'd12-(tmp+5'd12)%ten)/ten;
                    L_temp <= (tmp+5'd12)%ten;
                end
                Modetag_temp <=4'd12;
            end
            flag <= 0;
        end

        else
        begin
        if(select_EN)
        begin
            if(flicker_pos == current_pos)
            begin
                //设置标志位，防止一直调动
                if(! buttu)
                    up_flag<=1;
                if(! buttd)
                    down_flag<=1;
                //上调
                if(up_flag && buttu)
                begin
                    if((H_temp==2) && (L_temp==3))
                        {H_temp,L_temp} <= 8'h00;
                    else if((H_temp==2) && (L_temp<3))
                    begin
                        H_temp <= H_temp;
                        L_temp <= L_temp + 1;
                    end
                    else if(L_temp == 9)
                    begin
                        H_temp <= H_temp + 1;
                        L_temp <= 4'b0000;
                    end
                    else
                    begin
                        H_temp <= H_temp;
                        L_temp <= L_temp + 1;
                    end
                    up_flag <= 0;
                end
                //下调
                if(down_flag && buttd)
                begin
                    if((H_temp==0) && (L_temp==0))
                        {H_temp,L_temp} <= 8'h23;
                    else if(L_temp == 0)
                    begin
                        H_temp <= H_temp - 1;
                        L_temp <= 4'b1001;
                    end
                    else
                    begin
                        H_temp <= H_temp;
                        L_temp <= L_temp - 1;
                    end
                    down_flag <= 0;
                end
            end
        end
        end
    end

    //转化为对应进制时间
    always@(H_temp or L_temp or Mode)
    begin
        if(Mode==`Mode_24)
        begin
            H<=H_temp;
            L<=L_temp;
            Modetag<=4'd12;
        end
        else
        begin
            tmp_2= H_temp * ten+L_temp;
            if((tmp_2>=5'd1) && (tmp_2<=5'd11))
            begin
                H<=H_temp;
                L<=L_temp;
                Modetag<=4'd10;
            end
            else if(tmp_2==5'd0)
            begin
                H<=4'd1;
                L<=4'd2;
                Modetag<=4'd10;
            end
            else if(tmp_2==5'd12)
            begin
                H<=4'd1;
                L<=4'd2;
                Modetag<=4'd11;
            end
            else
            begin
                H<=(tmp_2-5'd12-(tmp_2-5'd12)%ten)/ten;
                L<=(tmp_2-5'd12)%ten;
                Modetag<=4'd11;
            end  
        end
    end
endmodule
秒表计时器模块

秒表模块分别调用秒表频率发生器，扫描频率发生器，秒表计数器，共阳数码管译码器，并对译码产生的段码依扫描频率逐个扫描。

因为秒表的计时精度为10ms,我们首先调用timediv产生100HZ频率作为秒表的基频，然后调用divider产生1KHZ的频率作为数码管的扫描频率，然后正确调用6个秒表计时器为秒表的6个时间显示位计时，为每一位都设置正确的模长，并且设置合适的使能信号把6个秒表计时器连接起来，然后调用digital_tube，输入计数器产生的计数值得到对应的段码，然后在扫描时钟下逐位扫描点亮数码管。

`include "para.v"

module timer(
    input clk,
    input timer_EN,
    input reset,pause,
    output reg [7:0] pos,
    output reg [6:0] seg
    );
    wire clk_100hz,clk_sweep;
    wire EN,SL_EN,SH_EN,ML_EN,MH_EN;
    wire [3:0] ms_one,ms_dec,s_one,s_dec,min_one,min_dec;
    wire [6:0] seg_cache[0:7];
    reg [7:0] tmp;
    timediv D0(clk,timer_EN,pause,clk_100hz);     //秒表频率发生器  100hz
    divider_sweep D1(clk,clk_sweep);            //扫描频率发生器  1Khz
    tmicnt MS0(clk_100hz,timer_EN,4'd10,reset,EN,ms_one);        
    tmicnt MS1(clk_100hz,timer_EN,4'd10,reset,(ms_one == 4'd9),ms_dec);
    tmicnt S0(clk_100hz,timer_EN,4'd10,reset,SL_EN,s_one);
    tmicnt S1(clk_100hz,timer_EN,4'd6,reset,SH_EN,s_dec);
    tmicnt M0(clk_100hz,timer_EN,4'd10,reset,ML_EN,min_one);
    tmicnt M1(clk_100hz,timer_EN,4'd6,reset,MH_EN,min_dec);        //秒表计数器  
    digital_tube T0(ms_one,seg_cache[0]);
    digital_tube T1(ms_dec,seg_cache[1]);
    digital_tube T2(s_one,seg_cache[2]);
    digital_tube T3(s_dec,seg_cache[3]);
    digital_tube T4(min_one,seg_cache[4]);
    digital_tube T5(min_dec,seg_cache[5]);        //共阳数码管译码器

    //设置使能信号
    assign EN = 1;                                //最低位永远使能有效
    //高位使能在次高位使能有效且低位将要进位的情况下有效
    assign SL_EN = (ms_one == 4'd9) && ((ms_dec == 4'd9));
    assign SH_EN = SL_EN && (s_one == 4'd9);
    assign ML_EN = SH_EN && (s_dec == 4'd5);
    assign MH_EN = ML_EN && (min_one == 4'd9);        

    assign seg_cache[7] = `segcode_5;
    assign seg_cache[6] = `segcode_None;    //模式显示标志
    initial
    begin
        tmp <= 8'b11111110;
    end
    //逐个扫描
    always @(posedge clk_sweep)
    begin
        tmp <= {tmp[6:0],tmp[7]};
        case(tmp)
            8'b11111110:seg <= seg_cache[0];
            8'b11111101:seg <= seg_cache[1];
            8'b11111011:seg <= seg_cache[2];
            8'b11110111:seg <= seg_cache[3];
            8'b11101111:seg <= seg_cache[4];
            8'b11011111:seg <= seg_cache[5];
            8'b10111111:seg <= seg_cache[6];
            8'b01111111:seg <= seg_cache[7];
        endcase
        pos <= tmp;
    end
endmodule
常规显示模块

显示模块分别调用了秒脉冲发生器，扫描频率发生器，计数器，调时模块，设置闹钟模块，七段共阳数码管译码器。实现正常同步计时并显示，整点报时，闹钟到点闪烁并可通过按键关闭。

正常显示：采用同步计时，将5个计数器级联起来，利用前级的进位信号生成后级的使能信号控制同步计时。

闹钟到点闪烁：flicker在设置闹钟的时/分与当前时间的时/分相同时有效，alarm在flicker无效即不在设定时间范围内闹钟不闪烁时重置为有效，若当前处于显示模块且confirm即关闭闹钟按键按下时，alarm无效，闪烁被强行关闭。闪烁原理与调试模块闪烁类似，当flicker和alarm同时有效时，在闪烁时钟clk_flicker的作用下，周期性的将段码置1，将数码管熄灭，表现出闪烁状态。

整点报时：int_time在当前时间的分和秒都为0，即为整点时有效，在秒脉冲的作用下，若为整点时刻，首先对当前的时间进行换算，换算成12小时进制，提取出上下午标志与12小时进制下的小时位。根据小时位点亮对应LED，根据标志位点亮三色灯区分上下午。

    reg [4:0] tmp_3;
    reg [2:0] tmp_rgb;

    assign select_EN = (~func[1])&(func[0]);
    assign alarm_EN = (func[1])&(~func[0]);

    //三种模式的输出选择器

    assign pos = alarm_EN ? alarm_pos : pos_temp;

    divider_100hz D0(clk,clk_1hz);    //产生1HZ的频率
   // defparam D0.out_freq = 1;
    divider_sweep D1(clk,clk_sweep);    //产生1KHZ的扫描频率
    divider_sweep D2(clk,clk_flicker);    //产生2HZ的闪烁频率
    defparam D2.out_freq_1 = 2;    

    //级联5个计时器
    count_10 S0(clk_1hz,SL_EN,select_EN,confirm,sone_cache,s_one);
    count_6 S1(clk_1hz,SH_EN,select_EN,confirm,sdec_cache,s_dec);
    count_10 M0(clk_1hz,ML_EN,select_EN,confirm,minone_cache,min_one);
    count_6 M1(clk_1hz,MH_EN,select_EN,confirm,mindec_cache,min_dec);    //获取分秒的个位数字
    count_Hour H0(clk_1hz,H_EN,Mode,select_EN,confirm,hdec_cache,hone_cache,Modetag_cache,hour_dec,hour_one,Mode_tag);    //获取上下午标志位及对应的小时数字


    //调用调时模块
    select_time U1(clk,Mode,buttl,buttr,buttu,buttd,select_EN,s_one,s_dec,min_one,min_dec,hour_one,hour_dec,Mode_tag,select_pos,select_seg,

    digital_tube T0(s_one,seg_cache[0]);
    digital_tube T1(s_dec,seg_cache[1]);
    digital_tube T2(min_one,seg_cache[2]);

    //设置使能信号

       display_pos <= 8'b11111110;
       tmp_rgb <= 3'b001;
       times <= 3'b0; 
       alarm <= 1'b1;
       RGB_led1=3'd0;
       RGB_led2=3'd0;
    end

    //整点报时模块
    always @(posedge clk_1hz)
    begin
        if(int_time)    //判断是否是整点
        begin
            times = 3'd6;    //设定为报时的秒数，相当于开启报时
            //如果是24小时制的时间则统一转化为12小时制的时间
            if (Mode_tag == 4'd12)
            begin
                tmp_2= hour_dec * ten+hour_one;

                    hdec_temp=hour_one;

                4'd10:

                    RGB_led1 = tmp_rgb;
                    tmp_rgb ={tmp_rgb[1:0],tmp_rgb[2]};
                end
                4'd11:
                begin
                    RGB_led2 = tmp_rgb;
                    tmp_rgb ={tmp_rgb[1:0],tmp_rgb[2]};

        end
        //不处于报时阶段，关闭所有Led和RGB Led

            RGB_led1=3'd0;
            RGB_led2=3'd0;
            tmp_rgb = 3'b001;
            led =12'b0000_0000_0000;
        end

    begin
        if(!flicker)
        begin
            alarm<=1'b1;
        end
        else
        begin
            if(confirm && display_EN)
            begin
                alarm<=1'b0;
            end

                else

                    display_seg <= 7'b1111111;
                else
                    display_seg <= seg_cache[3];
            end
            8'b11101111:
            begin
                if(flicker && (! clk_flicker) && alarm)
                    display_seg <= 7'b1111111;
                else
                    display_seg <= seg_cache[4];
            end
            8'b11011111:

                    display_seg <= 7'b1111111;

            8'b01111111:

                else
                    display_seg <= seg_cache[7];

调时模块

调时模块分别调用了扫描频率发生器，闪烁频率发生器，手动调时计数器，共阳七段数码管译码器。该模块实现将当前时刻传入调时，并且可以时，分，秒逐位进行调时。通过左右按键来选择被调位，选中的被调位进行闪烁突出显示；通过上下按键对选中的被调位调时，且在调时过程中可以切换12/24进制显示。

  1. 左右按键选择被调位：

    在1KHZ的扫描频率下同步检测左右按键的电平，并设立两个标志位left_flag, right_flag作为按键响应使能。当左/右按键空闲时，将标志位置1，开放按键响应；当检测到左/右按键被按下，即为高电平，且标志位是1时才进行向左/向右选择，即闪烁位置改变，响应按键，紧接着将标志位置0，关闭按键响应从而使按键此后直到松开前不会再被响应。待按键松开后，将标志位置1，重新开放按键响应。如此可以使高频同步电平检测模式下按键在按下时会且仅会被响应一次。

  2. 闪烁：

    在数码管依扫描时钟频率逐位扫描中，修改段码查找表：对于调时时间的每一位，判断是否应该当前位闪烁，并且是否是闪烁时钟的低电平。若是当前位闪烁且此时处于闪烁时钟低电平，则此位的段码置为全1，对应此时此位熄灭，其他情况赋值为正常段码，对应正常的动态扫描显示。

`include"para.v"
module select_time(
    input clk,sw15,buttl,buttr,buttu,buttd,
    input select_EN,
    input  [3:0] sone,
    input  [3:0] sdec,
    input  [3:0] minone,
    input  [3:0] mindec,
    input  [3:0] hone,
    input  [3:0] hdec,
    input  [3:0] Modetag,
    output reg [7:0] pos,
    output reg [6:0] seg,
    output  [3:0] sone_cache,
    output  [3:0] sdec_cache,
    output  [3:0] minone_cache,
    output  [3:0] mindec_cache,
    output  [3:0] hone_cache,
    output  [3:0] hdec_cache,
    output  [3:0] Modetag_cache
    );
    wire clk_sweep,clk_flicker;
    wire [6:0] seg_cache[0:7];
    wire [3:0] s_one,s_dec,min_one,min_dec,h_one,h_dec,Mode_tag;
    reg left_flag,right_flag;
    reg [3:0] flicker_pos;
    reg [7:0] tmp;

    divider_sweep D0(clk,clk_sweep);    //产生1KHZ扫描频率
    divider_sweep D1(clk,clk_flicker);    //产生2HZ闪烁频率
    defparam D1.out_freq_1 = 2;

    //调节时间部分
    //调节秒的个位
    manua_lcount C0(clk,select_EN,buttu,buttd,4'd10,flicker_pos,4'd0,sone,s_one);
    //调节秒的十位
    manua_lcount C1(clk,select_EN,buttu,buttd,4'd6,flicker_pos,4'd1,sdec,s_dec);
    //调节分的个位
    manua_lcount C2(clk,select_EN,buttu,buttd,4'd10,flicker_pos,4'd2,minone,min_one);
    //调节分的十位
    manua_lcount C3(clk,select_EN,buttu,buttd,4'd6,flicker_pos,4'd3,mindec,min_dec);
    //调节时
    manual_count24 C4(clk,select_EN,sw15,buttu,buttd,flicker_pos,4'd4,Modetag,hdec,hone,Mode_tag,h_dec,h_one);

    //共阳数码管译码器
    digital_tube T0(s_one,seg_cache[0]);
    digital_tube T1(s_dec,seg_cache[1]);
    digital_tube T2(min_one,seg_cache[2]);
    digital_tube T3(min_dec,seg_cache[3]);
    digital_tube T4(h_one,seg_cache[4]);
    digital_tube T5(h_dec,seg_cache[5]);
    digital_tube T7(Mode_tag,seg_cache[7]);

    assign seg_cache[6] = `segcode_None;

    initial
    begin
        flicker_pos <= 4'b0000;
        left_flag <= 1;
        right_flag <= 1;
        tmp <= 8'b11111110;
    end

    //扫描点亮每一位数码管 闪烁 or 常亮
    always @(posedge clk_sweep)
    begin
    if(select_EN)
    begin
        if(! buttl)
            left_flag<=1;
        if(! buttr)
            right_flag<=1;
        if(left_flag && buttl)
            begin
                flicker_pos <= (flicker_pos + 1) % 5;
                left_flag <= 0;
            end
        if(right_flag && buttr)
        begin
            if(flicker_pos == 0)
                flicker_pos <= 4;
            else
                flicker_pos <= flicker_pos - 1;
            right_flag <= 0;
        end
        tmp <= {tmp[6:0],tmp[7]};
        case(tmp)
            8'b11111110:
                begin
                    if(flicker_pos == 0 && (! clk_flicker))
                        seg <= 7'b1111111;
                    else
                        seg <= seg_cache[0];
                end
            8'b11111101:
                begin
                    if(flicker_pos == 1 && (! clk_flicker))
                        seg <= 7'b1111111;
                    else
                        seg <= seg_cache[1];
                end
            8'b11111011:
                begin
                    if(flicker_pos == 2 && (! clk_flicker))
                        seg <= 7'b1111111;
                    else
                        seg <= seg_cache[2];
                end
            8'b11110111:
                begin
                    if(flicker_pos == 3 && (! clk_flicker))
                        seg <= 7'b1111111;
                    else
                        seg <= seg_cache[3];
                end
            8'b11101111:
                begin
                    if(flicker_pos == 4 && (! clk_flicker))
                        seg <= 7'b1111111;
                    else
                        seg <= seg_cache[4];
                end
            8'b11011111:
                begin
                    if(flicker_pos == 4 && (! clk_flicker))
                        seg <= 7'b1111111;
                    else
                        seg <= seg_cache[5];
                end
            8'b10111111:
                begin
                        seg <= seg_cache[6];
                end
            8'b01111111:
                begin
                        seg <= seg_cache[7];
                end
        endcase
        pos <= tmp;
    end
    end

    //结果回传
    assign  sone_cache = s_one,
            sdec_cache = s_dec,
            minone_cache = min_one,
            mindec_cache = min_dec,
            hone_cache = h_one,
            hdec_cache = h_dec,
            Modetag_cache = Mode_tag;   

endmodule
闹钟模块

闹钟模块调用了扫描频率发生器，闪烁频率发生器，手动调时/分计数器，七段共阳数码管显示段码译码器。其中，闪烁与手动调时部分的逻辑与调时模块完全一致，此外，该模块还将设置的闹钟时间锁存起来，使得每次进入闹钟模块设置闹钟时都以上一次设置的时间为起始。

`include"para.v"
module set_alarm(
    input clk,sw15,buttl,buttr,buttu,buttd,
    input alarm_EN,
    output reg [7:0] pos,
    output reg [6:0] seg,
    output  [3:0] minone_cache,
    output  [3:0] mindec_cache,
    output  [3:0] hone_cache,
    output  [3:0] hdec_cache,
    output  [3:0] Modetag_cache
    );
    reg  [3:0] minone;
    reg  [3:0] mindec;
    reg  [3:0] hone;
    reg  [3:0] hdec;
    reg  [3:0] Modetag;
    initial
    begin
       minone <= 4'd0;
       mindec <= 4'd0;
       hone <= 4'd0;
       hdec <= 4'd0;
       Modetag <= 4'd12;
    end
    wire clk_sweep,clk_flicker;
    wire [6:0] seg_cache[0:7];
    wire [3:0] min_one,min_dec,h_one,h_dec,Mode_tag;
    reg left_flag,right_flag;
    reg [3:0] flicker_pos;
    reg [7:0] tmp;

    divider_sweep D0(clk,clk_sweep);    //产生1KHZ扫描频率
    divider_sweep D1(clk,clk_flicker);    //产生2HZ的闪烁频率
    defparam D1.out_freq_1 = 2;

    //手动调时间部分
    //调分个位
    manua_lcount C0(clk,alarm_EN,buttu,buttd,4'd10,flicker_pos,4'd0,minone,min_one);
    //调分十位
    manua_lcount C1(clk,alarm_EN,buttu,buttd,4'd6,flicker_pos,4'd1,mindec,min_dec);
    //调时
    manual_count24 C2(clk,alarm_EN,sw15,buttu,buttd,flicker_pos,4'd2,Modetag,hdec,hone,Mode_tag,h_dec,h_one);

    //共阳数码管译码器
    digital_tube T2(min_one,seg_cache[0]);
    digital_tube T3(min_dec,seg_cache[1]);
    digital_tube T4(h_one,seg_cache[2]);
    digital_tube T5(h_dec,seg_cache[3]);
    digital_tube T7(Mode_tag,seg_cache[4]);

    //标志位段码指定
    assign seg_cache[5] = `segcode_None;
    assign seg_cache[6] = `segcode_L;
    assign seg_cache[7] = `segcode_A;

    initial
    begin
        flicker_pos <= 4'b0000;
        left_flag <= 1;
        right_flag <= 1;
        tmp <= 8'b11111110;
    end

    //扫描模块
    always @(posedge clk_sweep)
    if(alarm_EN)
    begin
        //移动调时位
        if(! buttl)
            left_flag<=1;
        if(! buttr)
            right_flag<=1;
        if(left_flag && buttl)
            begin
                flicker_pos <= (flicker_pos + 1) % 3;
                left_flag <= 0;
            end
        if(right_flag && buttr)
        begin
            if(flicker_pos == 0)
                flicker_pos <= 2;
            else
                flicker_pos <= flicker_pos - 1;
            right_flag <= 0;
        end

        //扫描点亮每一位数码管 闪烁 or 常亮
        tmp <= {tmp[6:0],tmp[7]};
        case(tmp)
            8'b11111110:
                begin
                    if(flicker_pos == 0 && (! clk_flicker))
                        seg <= 7'b1111111;
                    else
                        seg <= seg_cache[0];
                end
            8'b11111101:
                begin
                    if(flicker_pos == 1 && (! clk_flicker))
                        seg <= 7'b1111111;
                    else
                        seg <= seg_cache[1];
                end
            8'b11111011:
                begin
                    if(flicker_pos == 2 && (! clk_flicker))
                        seg <= 7'b1111111;
                    else
                        seg <= seg_cache[2];
                end
            8'b11110111:
                begin
                    if(flicker_pos == 2 && (! clk_flicker))
                        seg <= 7'b1111111;
                    else
                        seg <= seg_cache[3];
                end
            8'b11101111:
                begin
                    seg <= seg_cache[4];
                end
            8'b11011111:
                begin
                    seg <= seg_cache[5];
                end
            8'b10111111:
                begin
                        seg <= seg_cache[6];
                end
            8'b01111111:
                begin
                        seg <= seg_cache[7];
                end
        endcase
        pos <= tmp;
    end

    //将每次设定闹钟结果送进寄存器，以便下次可以从当前设定闹钟时间开始调节
    always@(min_one or min_dec or h_one or h_dec or Mode_tag)
    begin
       minone <= min_one;
       mindec <= min_dec;
       hone <= h_one;
       hdec <= h_dec;
       Modetag <= Mode_tag;
    end

    //结果回传
    assign  minone_cache = min_one,
            mindec_cache = min_dec,
            hone_cache = h_one,
            hdec_cache = h_dec,
            Modetag_cache = Mode_tag;

endmodule

实验结果

在Vivado中保存设计文件，生成比特流。打开硬件管理器连接开发板，将生成的比特流下载到开发板中，可以看到数码管上正常计时并显示时间，拨动显示进制开关sw15可以切换24/12小时进制显示

拨动模式选择开关sw1和sw2切换至调时模式，可以看到调时初值即为当前时刻值，同时秒个位闪烁，按下左右按键可以切换不同位闪烁，按下上下按键调整闪烁位的数值，此时拨动显示进制开关sw15依旧可以切换24/12小时进制显示，按下中间按键后拨动模式选择开关回到显示模式，可以看到数码管上显示的是已经调好的时间

拨动模式选择开关切换至闹钟模式，可以看到最左侧数码管显示“AL”表示当前为闹钟模式，分个位闪烁，按下左右上下按键可以切换不同位并调整时间。闹钟设置完毕后切换回显示模式，当当前时间时间到设置的闹钟时间时，时分秒数码管开始闪烁，表示闹钟响起。按下中间按键后停止闪烁(不按则在1分钟后自动停止)。

拨动模式选择开关切换至秒表模式，可以看到最左侧数码管显示“S”表示当前为秒表模式，初始值为00:00:00。按下中间按键，秒表计时开始，精度为10ms，再次按下中间按键，计时暂停，再次按下，计时继续。按下下按键，秒表复位清零。

切换模式至显示模式，当当前时间到达整点时，对应12小时进制的小时数的LED点亮，同时对应上下午的三色灯点亮，6秒后熄灭。

实验感想

这个实验是由我们两个人一起合作完成的，所以我们在进行代码开发时需要更加注重结构设计和模块划分，这个过程也锻炼了我们的分工合作能力，并且我们会相互检查各自负责测试部分的模块代码，这也在一定程度上保证了我们的项目程序要更加稳定和健壮，并且我们的设计是从用户使用体验出发的，很多细微之处的设计也倾注了我们很大的心血。

这次实验不仅极大提高了我们对verilog项目的开发能力，还极大锻炼了我们的分工合作能力，收获很大。
